# Energy Cost Add-ons

Home Assistant app/add-on repository for imported electricity cost statistics.

## Apps

- `energy_cost_importer`: imports calculated electricity cost statistics for the Energy dashboard.

## Local installation

Copy `energy_cost_importer` to `/addons/energy_cost_importer` on the Home Assistant host, reload the Add-on Store, and install `Energy Cost Importer`.

## Repository installation

Publish this folder as a Git repository and add the repository URL in the Home Assistant Add-on Store.
